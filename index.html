UrlSherif

UrlSherif é uma ferramenta Python que usa Tkinter, Requests e BeautifulSoup para varrer páginas da web. Ela detecta ameaças, monitora estatísticas de acesso e tem uma interface gráfica intuitiva para análise de URLs.

Registra-Click

Este projeto é um sistema em JavaFX para gerenciar registros de funcionários, permitindo operações como retirada e devolução de patrimônio, com geração automática de relatórios em Excel.

WebMine

WebMine é uma ferramenta em Python que extrai dados e arquivos de sites de forma prática e rápida. Ideal para quem precisa coletar informações diretamente da web, o WebMine facilita a extração de conteúdo, convertendo páginas online em dados.
